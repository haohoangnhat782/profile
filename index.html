Introduction

Checkers is a quite popular game because the rules are simple and do not require many playing tools, you only need a piece of paper with a grid and a pen. That's why this game is even more popular in Vietnam. student world. Checkers is a 2-player fighting game: Fighting game (twoagent, conflicting game): Including 2 players, each opponent will try to win against the other opponent in a finite number of moves, each move Created based on any state of the match. If after a limited number of moves, no one has won, then it is considered a draw. In addition, information about the match is completely known (perfect information) to both opponents.
